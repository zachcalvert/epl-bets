## API Reference Notes

### football-data.org v4

- Auth: `X-Auth-Token: {key}` header
- Rate limit: 10 requests/minute (free tier)
- EPL competition code: `PL`
- Match statuses: `SCHEDULED`, `TIMED`, `IN_PLAY`, `PAUSED`, `FINISHED`, `POSTPONED`, `CANCELLED`, `SUSPENDED`
- Season format: 4-digit start year (e.g., `2025` for 2025-26)
- Docs: https://docs.football-data.org/general/v4/index.html

### The Odds API v4

- Auth: `apiKey` query parameter
- Free tier: 500 requests/month
- Sport key: `soccer_epl`
- Markets: `h2h` (1X2), `spreads`, `totals`
- Regions: `us`, `uk`, `eu`, `au`
- Tracks remaining credits in response headers (`x-requests-remaining`, `x-requests-used`)
- Docs: https://the-odds-api.com/liveapi/guides/v4/
